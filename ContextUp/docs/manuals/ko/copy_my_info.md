# Copy My Info (내 정보 복사)

## 소개
자주 사용하는 텍스트 정보(계좌번호, 주소, 이메일 등)를 등록해두고 한 클릭으로 빠르게 복사합니다.

## 주요 기능
- **정보 등록**: 자주 쓰는 텍스트 저장
- **빠른 복사**: 한 클릭으로 클립보드에
- **카테고리 관리**: 그룹별 정리
- **암호화 저장**: 민감 정보 보호

## 사용법

### 정보 등록
1. 트레이 메뉴 → **Copy My Info → 관리** 선택
2. **+ 추가** 버튼 클릭
3. 정보 입력:
   - 이름 (예: "내 계좌")
   - 내용 (예: "123-456-789012")
4. 저장

### 정보 복사
1. 트레이 메뉴 → **Copy My Info**
2. 저장된 항목 클릭
3. 클립보드에 즉시 복사됨

## 저장 가능 정보 예시
- 은행 계좌번호
- 집/회사 주소
- 이메일 주소
- 전화번호
- 자주 쓰는 문구

## 보안
- 로컬에만 저장 (클라우드 업로드 없음)
- JSON 파일로 저장 (userdata/copy_my_info.json)
