# Finder (파인더)

## 소개
고속 해싱 알고리즘을 사용하여 중복 파일, 대용량 파일, 빈 폴더를 효율적으로 탐색하고 관리하는 유틸리티입니다.

## 주요 기능

### 중복 파일 찾기
- **xxHash 기반 고속 스캔**: 파일 이름이 달라도 내용이 동일한 파일 감지
- **그룹별 표시**: 동일 파일들을 그룹으로 묶어서 표시
- **원본 보존**: 각 그룹에서 보존할 파일 선택 가능

### 대용량 파일 찾기
- 지정 크기 이상의 파일만 필터링
- 크기순 정렬로 용량 확보 대상 빠르게 확인

### 빈 폴더 정리
- 재귀적으로 빈 하위 디렉토리 탐색
- 일괄 삭제로 불필요한 폴더 구조 정리

## 사용법

1. 폴더를 선택하고 우클릭 → **System → Finder** 실행
2. 또는 단축키 `Alt+Space`로 빠르게 실행
3. 검색 모드 선택 (중복/대용량/빈폴더)
4. 스캔 완료 후 처리할 항목 선택
5. **Delete** 또는 **Move to Trash**로 정리

## 단축키
- `Alt+Space`: Finder 바로 실행
- `Ctrl+A`: 전체 선택
- `Delete`: 선택 항목 삭제
