# Sequence Renumber (시퀀스 재번호)

## 소개
이미지 시퀀스의 프레임 번호를 새로운 시작점이나 패턴으로 재정렬합니다.

## 주요 기능
- **시작 번호 변경**: 0001부터 또는 특정 번호부터
- **자릿수 조정**: 4자리, 5자리 등 패딩 변경
- **간격 조정**: 1씩 또는 특정 간격으로
- **프레임 오프셋**: 전체 번호 시프트

## 사용법

1. 시퀀스 폴더 또는 이미지들 선택 후 우클릭
2. **Sequence → Sequence Renumber** 선택
3. 옵션 설정:
   - **시작 번호**: 새 시퀀스 시작 번호
   - **자릿수**: 패딩 자릿수 (4→0001)
   - **간격**: 프레임당 증가값
4. 미리보기 확인
5. 적용
