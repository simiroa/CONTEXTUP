# Sequence Missing Frames (누락 프레임 찾기)

## 소개
이미지 시퀀스에서 누락된 프레임을 자동으로 탐지하고 목록을 생성합니다.

## 사용법

1. 시퀀스 폴더 또는 첫 번째 이미지 선택 후 우클릭
2. **Sequence → Sequence Missing Frames** 선택
3. 스캔 결과 확인

## 출력 정보
- 누락 프레임 번호 목록
- 연속 누락 구간 표시
- 재렌더 명령어 생성 (선택)

## 예시 결과
```
누락 프레임: 15개
- 0045
- 0178-0182 (5프레임)
- 0234-0243 (10프레임)

재렌더 범위: 45,178-182,234-243
```

## 활용 예시
- 렌더팜 실패 프레임 확인
- 재렌더 목록 자동 생성
- 시퀀스 무결성 검증

## 지원 시퀀스 패턴
- `image_0001.png` (넘버링)
- `shot_001_v02.exr` (복합)
- `frame.1234.jpg` (dot 구분)
