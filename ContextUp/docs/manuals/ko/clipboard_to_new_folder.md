# Paste to New Folder (새 폴더에 붙여넣기)

## 소개
클립보드에 복사된 파일들을 새 폴더를 생성하고 그 안에 붙여넣습니다.

## 사용법

1. 파일들 복사 (Ctrl+C)
2. 대상 위치에서 우클릭
3. **Clipboard → Paste to New Folder** 선택
4. 폴더 이름 입력
5. 새 폴더 생성 + 파일 붙여넣기 완료

## 동작 예시
```
클립보드: file1.png, file2.png, file3.png

결과:
└── 새폴더/
    ├── file1.png
    ├── file2.png
    └── file3.png
```

## 활용 예시
- 다운로드 파일 정리
- 프로젝트별 폴더 생성 및 파일 이동
- 작업 파일 그룹화

## 기능
- **새 폴더 자동 생성**: 이름 입력 후 즉시 생성
- **붙여넣기 통합**: 폴더 생성과 붙여넣기를 한 번에
- **이름 제안**: 첫 번째 파일명 기반 제안
