# Manager (관리자)

## 소개
ContextUp의 모든 설정을 제어하는 통합 관리 패널입니다. 메뉴 커스터마이징, API 키 관리, 의존성 설치 상태 확인 등을 수행합니다.

## 주요 기능

### 탭별 기능
| 탭 | 기능 |
|---|------|
| **Items** | 우클릭 메뉴 항목 표시/숨김 설정 |
| **Settings** | API 키, 외부 도구 경로 설정 |
| **Dependencies** | 패키지/모델 설치 상태 확인 |
| **Logs** | 실행 로그 및 오류 확인 |

### 설정 항목
- **Gemini API Key**: AI 텍스트 분석용
- **Ollama Path**: 로컬 LLM 경로
- **ComfyUI Path**: 생성형 AI 연동
- **Blender Path**: 3D 도구 연동
- **FFmpeg Path**: 비디오 처리 (자동 설치)

## 사용법

1. 트레이 아이콘 더블클릭 또는 메뉴에서 **Manager** 선택
2. 원하는 탭 선택
3. 설정 변경 후 **Save** 클릭

## 메뉴 커스터마이징
- Items 탭에서 각 기능 표시/숨김 토글
- 자주 쓰는 기능만 표시하여 메뉴 간소화
- 카테고리별 일괄 숨김 가능
