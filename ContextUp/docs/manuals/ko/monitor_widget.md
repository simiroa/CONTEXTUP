# Monitor Widget

## 소개
CPU, GPU(NVIDIA), RAM, VRAM, 디스크, 네트워크, 로컬 서버 현황을 실시간으로 감시하는 데스크탑 위젯입니다.

## 주요 기능

### 사이드바 (C/R/G/V/D/N/S)
| 항목 | 설명 |
|------|-----|
| **C** | CPU 사용량 (%) |
| **R** | RAM 사용량 (%) |
| **G** | GPU 사용량 (NVIDIA) |
| **V** | VRAM 사용량 |
| **D** | 디스크 I/O 속도 (MB/s) |
| **N** | 네트워크 I/O 속도 |
| **S** | 활성 로컬 서버 포트 수 |

### 상세 패널
각 항목을 클릭하면 상세 정보가 확장됩니다:
- **프로세스 목록**: CPU/RAM/GPU별 상위 사용 프로세스
- **디스크 탭**: 드라이브별 사용량 및 폴더 바로가기
- **네트워크 탭**: 속도 테스트, IP 확인, 인터넷 토글
- **서버 탭**: 활성 포트 목록, 브라우저 열기, 프로세스 종료

## 사용법

1. 트레이 메뉴에서 **Monitor Widget** 실행
2. 사이드바 항목 클릭으로 상세 패널 확장/축소
3. 위젯 드래그로 화면 원하는 위치로 이동 (모서리 스냅 지원)
4. 상단 슬라이더로 투명도 조절
5. "Top" 체크박스로 항상 위 모드 토글

## 단축 기능
- **Speed Test**: 네트워크 다운로드 속도 측정
- **인터넷 On/Off**: 네트워크 어댑터 활성화/비활성화
- **프로세스 종료**: × 버튼으로 해당 프로세스 강제 종료
- **서버 열기**: 🔗 버튼으로 localhost 브라우저 열기

## 시스템 요구사항
- Windows 10/11
- NVIDIA GPU (GPU/VRAM 모니터링용, 선택)
- 관리자 권한 (일부 기능)
