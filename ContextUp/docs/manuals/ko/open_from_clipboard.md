# Open from Clipboard (경로 열기)

## 소개
클립보드에 복사된 경로를 감지하여 해당 위치를 바로 탐색기로 엽니다.

## 사용법

1. 파일/폴더 경로를 클립보드에 복사
2. 단축키 `Ctrl+Alt+V` 또는 트레이 메뉴에서 실행
3. 해당 경로가 탐색기에서 열림

## 지원 형식
- **로컬 경로**: `C:\Users\Documents\file.txt`
- **UNC 경로**: `\\Server\Share\folder`
- **URL**: `file:///C:/path/to/file`

## 활용 예시
- 채팅/이메일에서 받은 경로 바로 열기
- 로그 파일의 경로 복사 후 이동
- 문서에 적힌 경로 탐색

## 단축키
- `Ctrl+Alt+V`: 경로 열기 바로 실행

## 기능
- **경로 자동 감지**: 클립보드 내용 분석
- **존재 여부 확인**: 경로가 없으면 경고
- **부모 폴더 열기**: 파일인 경우 폴더를 열고 파일 선택
