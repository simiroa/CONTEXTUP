# Sequence Analyze (시퀀스 분석)

## 소개
이미지 시퀀스 폴더를 분석하여 프레임 범위, 누락 프레임, 파일 정보를 한눈에 보여줍니다.

## 분석 정보

| 항목 | 설명 |
|-----|-----|
| **프레임 범위** | 시작~끝 프레임 번호 |
| **총 프레임 수** | 전체 이미지 개수 |
| **누락 프레임** | 빠진 프레임 번호 목록 |
| **해상도** | 이미지 가로×세로 |
| **포맷** | 파일 확장자 |
| **용량** | 전체/평균 파일 크기 |

## 사용법

1. 시퀀스 폴더 또는 첫 번째 이미지 선택 후 우클릭
2. **Sequence → Sequence Analyze** 선택
3. 분석 결과 확인

## 출력 예시
```
시퀀스: render_####.exr
범위: 0001-0240
총 프레임: 238/240 (2개 누락)
누락: [0045, 0178]
해상도: 1920×1080
용량: 4.2 GB (평균 18MB/프레임)
```

## 활용 예시
- 렌더팜 출력 확인
- 누락 프레임 재렌더 목록 생성
- 시퀀스 정보 빠른 확인
