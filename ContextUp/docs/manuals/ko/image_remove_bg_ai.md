# Remove Background AI (AI 배경 제거)

## 소개
RMBG 딥러닝 모델을 사용하여 이미지의 배경을 자동으로 감지하고 제거합니다. 투명 PNG로 저장됩니다.

## 주요 기능
- **자동 피사체 감지**: 사람, 동물, 물체 등 자동 인식
- **정밀 마스킹**: 머리카락 등 세부 영역까지 처리
- **배치 처리**: 여러 이미지 일괄 처리
- **투명 배경 출력**: PNG 형식으로 저장

## 사용법

1. 이미지 파일(들) 선택 후 우클릭
2. **AI → Remove Background** 선택
3. AI 처리 완료 대기 (이미지당 2-5초)
4. 결과 확인

## 출력 파일
- `원본파일명_nobg.png`: 투명 배경 이미지

## 지원 포맷
- **입력**: PNG, JPG, JPEG, WebP, BMP
- **출력**: PNG (투명 배경)

## 활용 예시
- 제품 사진 배경 제거
- 증명사진 배경 교체
- 합성용 이미지 준비
- 썸네일/배너 제작

## 시스템 요구사항
- NVIDIA GPU 권장
- VRAM 2GB 이상

## 의존성
- rembg / BiRefNet
- PyTorch
