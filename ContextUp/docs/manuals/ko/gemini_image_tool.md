# Gemini Image Tool (Gemini AI 이미지 분석)

## 소개
Google Gemini Vision API를 사용하여 이미지의 내용을 분석하고 설명, 태그, 데이터 추출 등을 수행합니다.

## 주요 기능
- **이미지 설명**: 이미지 내용을 자연어로 설명
- **태그 생성**: 자동 키워드/태그 추출
- **텍스트 추출**: 이미지 내 텍스트 OCR
- **질문 응답**: 이미지에 대한 질문 가능
- **데이터 추출**: 표, 차트에서 데이터 추출

## 사용법

1. 이미지 파일 선택 후 우클릭
2. **AI → Gemini Image Tool** 선택
3. 질문 입력 또는 기본 분석 선택:
   - **Describe**: 이미지 설명
   - **Extract Text**: 텍스트 OCR
   - **Generate Tags**: 태그 생성
   - **Custom**: 직접 질문 입력
4. 분석 결과 확인

## 설정 요구사항
- **Gemini API Key**: Manager → Settings에서 설정
- 인터넷 연결 필요

## 활용 예시
- 제품 이미지 설명 자동 생성
- 스캔 문서 텍스트 추출
- 사진 자동 태깅
- 차트/그래프 데이터 추출
