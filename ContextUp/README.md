## ðŸš€ Quick Start (English Summary)

### 1. Installation
Run `install.bat` at the root directory.
This will set up the embedded Python environment and install necessary dependencies.

### 2. Usage
- **Right-Click Menu**: Access various tools directly from your context menu.
- **Manager**: Run `manager.bat` to toggle features and manage settings.
- **Quick Menu**: Press `Ctrl+Shift+C` anytime to bring up the quick launcher.
