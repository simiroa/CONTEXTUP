# Icon Generation Prompts

Use these prompts with Gemini Image Generation or other AI tools to create high-quality icons for ContextUp.

## General Style
**Style:** Modern, Minimalist, 3D, Glassmorphism, Vibrant Colors, Rounded Corners.
**Background:** Transparent or Solid Circle.
**Perspective:** Isometric or Front-facing.

## 1. Manager GUI (manager.ico)
**Prompt:** A 3D isometric icon of a futuristic control panel or dashboard with sliders and switches. The main color is dark blue (#2C3E50). Glassmorphism effect on the panel. High quality, 4k render.

## 2. Gemini Image Tools (gemini.ico)
**Prompt:** A glowing, magical crystal or brain structure emitting purple and blue light. Represents AI intelligence. Main color is purple (#8E44AD). 3D render, subsurface scattering.

## 3. Subtitle Tools (subtitle.ico)
**Prompt:** A speech bubble with "CC" text inside, floating in 3D space. The bubble is orange (#E67E22) with a glossy finish. Cinematic lighting.

## 4. Texture Tools (texture.ico)
**Prompt:** A 3D cube with different textures on each face (wood, metal, stone). Represents material processing. Main color is green (#27AE60). Sharp details.

## 5. Flatten Directory (flatten.ico)
**Prompt:** A 3D folder being compressed or a stack of papers turning into a single sheet. Represents flattening. Main color is red (#C0392B). Dynamic motion blur.

## 6. Archive Manager (archive.ico)
**Prompt:** A golden cardboard box or a futuristic safe. Represents storage and archiving. Main color is yellow/gold (#F1C40F). Metallic texture.

## 7. Tray Agent (tray.ico)
**Prompt:** A small, sleek robot or a floating orb that looks like a helper. Main color is light blue (#3498DB). Friendly appearance.

## 8. YouTube Downloader (youtube.ico)
**Prompt:** A red play button with a downward arrow, stylized in 3D. Main color is red (#E74C3C). Glossy plastic material.

## 9. Real-time Translator (translator.ico)
**Prompt:** Two speech bubbles with flags or language symbols (A/æ–‡) overlapping. Main color is teal (#1ABC9C). Glassy transparency.

## 10. Mayo Tools (mayo.ico)
**Prompt:** A stylized 3D CAD model or a blueprint roll. Represents 3D engineering. Main color is grey/silver (#95A5A6). Technical look.
