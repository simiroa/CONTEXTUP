# ContextUp 시작하기 🚀

Windows 우클릭 메뉴를 강력하게 확장하는 도구입니다.

---

## 📋 시스템 요구사항

- **OS**: Windows 10/11
- **Python**: 3.10 이상 (첫 설치 시만 필요, 이후 내장 Python 사용)
- **권장**: NVIDIA GPU (AI 기능 사용 시)

---

## 📥 설치 방법

### 1단계: 설치 실행
`install.bat`을 **더블클릭**하세요!

```
📁 HG_context_v2
   └── 🔧 install.bat  ← 이것을 실행!
```

### 2단계: 설치 옵션 선택
설치 중 아래 옵션이 표시됩니다:

| 옵션 | 설명 |
|------|------|
| **Minimal** | 핵심 기능만 (빠름, ~100MB) |
| **Standard** | 일반 기능 포함 (권장, ~500MB) |
| **Full** | AI 기능 포함 (느림, ~8GB+) |
| **Custom** | 원하는 기능 직접 선택 |

> 💡 **팁**: 처음이라면 **Standard**를 추천드립니다!

### 3단계: 설치 완료
설치가 끝나면 **Manager가 자동으로 실행**됩니다.
- 우클릭 메뉴는 자동 등록됩니다
- Manager에서 기능을 켜고/끄고 **Save Changes**로 저장하세요
- 트레이 아이콘 우클릭 또는 `Ctrl+Shift+C`로 빠른 메뉴를 열 수 있습니다

---

## 🗑️ 제거 방법

### 방법 1: uninstall.bat 실행
`uninstall.bat`을 **더블클릭**하면 제거가 시작됩니다.

```
📁 HG_context_v2
   └── 🔧 uninstall.bat  ← 이것을 실행!
```

### 방법 2: Manager에서 제거
1. 시스템 트레이 아이콘 우클릭 → **Manager** 선택
2. 설정 탭에서 **Uninstall** 클릭

제거 시 선택 가능한 옵션:
- ✅ 레지스트리 정리 (우클릭 메뉴 제거)
- ✅ 설정 파일 삭제
- ⬜ 외부 도구 삭제 (ComfyUI, Blender 등 - 선택사항)

## ❓ 자주 묻는 질문

### Q: 우클릭 메뉴가 안 보여요
**A**: `ContextUpManager.bat`을 실행하고 **Re-register Menu** 버튼을 클릭하세요.

### Q: 특정 기능이 작동하지 않아요
**A**: 해당 기능의 의존성이 설치되지 않았을 수 있습니다. Manager에서 확인하세요.

### Q: AI 기능을 나중에 추가하고 싶어요
**A**: `install.bat`을 다시 실행하면 추가 설치가 가능합니다.
